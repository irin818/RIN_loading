# RIN Live2D References

This folder stores original visual references for the RIN Live2D model.

Current chat references:

1. Full reference board
   - Front half-body view
   - Turnaround front/side/back
   - Expression grid
   - Detail closeups
   - Live2D layer breakdown reference
   - Design notes, keywords, logo/signature

2. Clean reference board
   - Large front bust portrait
   - Turnaround front/side/back
   - Expression strip
   - Detail strip
   - Color palette
   - Costume details

The actual image attachments are visible in the chat, but the original image files are not yet stored on disk. For production work, place the original PNG/PSD files here with stable names such as:

- `rin_reference_full_board.png`
- `rin_reference_clean_board.png`
- `rin_source_layered.psd`

Do not overwrite source references. Add new versions with version suffixes such as `_v02`.
